Comparativa de protocolos de routing

El routing es el proceso mediante el cual los routers determinan el camino más eficiente para enviar los paquetes de datos desde una dirección IP de origen hasta una dirección IP de destino. Esto incluye redes internas y externas, garantizando que los datos lleguen de manera confiable y eficiente.

Los protocolos de routing permiten que los routers intercambien información y actualicen sus tablas de rutas según la topología de la red, evitando bucles y optimizando el tráfico.

Routing Estático:
Configurado manualmente. Sencillo y seguro, pero no se adapta a fallos automáticamente.

Routing Dinámico:
Ajuste automático según cambios en la red, ideal para redes medianas/grandes.

  • Tipo: Vector de distancia
  • Métrica: Número de saltos (máx 15)
  • Convergencia: Lenta
  • Escalabilidad: Adecuada para redes pequeñas
  • Ventajas: Fácil de configurar, bajo consumo de recursos
  • Desventajas: No adecuado para redes grandes

  • Tipo: Estado de enlace
  • Métrica: Costo de los enlaces
  • Convergencia: Rápida
  • Escalabilidad: Media-alta
  • Ventajas: Evita bucles, eficiente
  • Desventajas: Configuración más compleja
